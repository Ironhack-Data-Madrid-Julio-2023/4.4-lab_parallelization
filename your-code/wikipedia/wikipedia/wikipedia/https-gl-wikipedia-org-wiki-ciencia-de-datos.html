A ciencia dos datos, tamén coñecida como ciencia de datos ou data science en inglés,[1] é un campo interdisciplinar de métodos, procesos e sistemas científicos para extraer información e coñecemento dos datos. Usando datos estruturados ou non estruturados (véxase minaría de datos), a ciencia de datos é un campo da ciencia que une a estatística, a informática e outros campos do coñecemento co obxectivo final de comprender e modelar un fenómeno.[2] Dentro desta área podemos atopar outros subcampos como a aprendizaxe automática, a aprendizaxe profunda, a clasificación, a análise de clúster, a minaría de datos, as bases de datos ou a visualización de datos.[3]

A ciencia de datos é unha nova metodoloxía enfocada a converter os datos en accións. Así, en lugar de basear as decisións na intuición, este campo ofrece ferramentas para optimizar a toma de decisións mediante o coñecemento e a análise de resultados. A ciencia de datos é unha ferramenta para informar, non para explicar.

Polo xeral, os pasos a seguir nestes procesos son: facerse unha pregunta, obter os datos, procesalos (límpalos, agrupalos, seleccionalos...), analizalos e, finalmente, tomar unha decisión e actuar.
